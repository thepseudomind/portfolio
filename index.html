recent projects

eazy food

An e-commerce solution for users to buy food items from the comfort of their devices which i created using HTML5, CSS3, SASS and JavaScript. Also made use of Firebase as the backend so the users could create accounts, login, add food items to favorites, implement an efficient CRUD functionality in the shopping cart.

eazy rent

A project built for tenants or home seekers looking to lease a home on a monthly or yearly basis. Built with HTML5, CSS3, ReactJS, NextJS for the development of the frontend. Also used JSON-Server as the temporary server so the users could create and delete listings, add banks for payments, view and edit the banks etc.
